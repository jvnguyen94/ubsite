There are three tab-delimited files: 

1) ptm.txt -- PTM information
	ptm_type 
	source 
	substrate_UniProtAC 
	substrate_genename
	organism
	site 
	enzyme_UniProtAC 
	enzyme_genename
	note 
	pmid

2) score.txt -- score information
        substrate_UniProtAC
        site
        enzyme_UniProtAC
        ptm_type
        score

Note: The score is calculated based on all the evidence sources as shown on the website.
  
3) protein.txt -- Protein Information
	UniProtAC
	UniProtID
	protein_name
	genename
	organism
	PRO_ID
	UniProtKB-SwissProt_or_TrEmbl
